This Project was launched to be able to link my Verisure France alarm and my Homey Pro 2023.

I do not intend to use Verisure sensors or cameras, but being able to change the status of the alarm device :
  - With a schedule action     : Arm the alarm at a specific hour, disarm it
  - With a canditionnal action : When everybody's out then arm the alarm, When someone come back disarm it..

I've created this project using other gitbubs to proceed to a reverse engineering of apps/plugins running on Jeedom/HA/...
I've not written even a line of code, but used Claude to write all of it (Vide coding) under my supervision.

Have a look to the Documentation folder where you will find the user manual and the technical app documentation, written by Claude also, under my supervision
